# 精灵图
 为了有效减少服务器接收和发送请求的次数 提高页面加载速度
 出现了CSS精灵技术 也称（CSS Sprites CSS雪碧）
 
 核心原理 ：将网页中的一些小背景图像整合到一张大图中
 这样服务器只需要一次请求就可以了.
 
 使用方式
 给小盒子添加背景图片 然后使用background-position移动背景图片 让需要的图标和盒子对齐
 （需要精确坐标X Y 网页中坐标往上/往左是负值）‘
 position可以连写 例
 
 background：url（images/shiro1.png） -182px 0;
